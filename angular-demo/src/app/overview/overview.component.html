Welcome to Angular Overview

This learning application is designed to help you bridge your knowledge from React/Blazor to modern Angular 18.

What's in this guide?

Use the sidebar to navigate through the core concepts of Angular. Each lesson contains rich explanations, live interactive demos, and code snippets detailing the React equivalent.

📝 Variables & Inputs Learn about Normal Variables, Signals, and Inputs (like Blazor Parameters or React Props). 🧩 Components vs Pages Learn the conceptual difference between a Smart Page Component and a Reusable Dumb Component. 🖱️ Event Listeners Learn how to bind to clicks, keyboard events, mouse events, form inputs, and how to emit custom output events. 🌐 Services & HTTP Learn how to fetch data from APIs using Angular's HttpClient and Dependency Injection. 🔢 Signals (Counter) Learn how to manage state reactively using Angular Signals (the modern equivalent to React's useState).

Core Philosophy

Unlike React which is just a UI library, Angular is a full-fledged framework. It provides built-in solutions for Routing, Forms, HTTP requests, and Dependency Injection right out of the box.

In Modern Angular (v14+), you use Standalone Components, which means you no longer need complex NgModules. This makes building in Angular feel much closer to React!
