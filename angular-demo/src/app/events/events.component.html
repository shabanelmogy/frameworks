Event Listeners in Angular

In Angular, you bind to DOM events using parentheses (event)="method()". This is equivalent to native addEventListener.

1. Basic Click Event (click)

The most common event. Wrap the target event in parentheses.

You clicked the button: {{ clickCount }} times.

2. Passing the Event Object ($event)

Pass the special $event variable to get details about the event.

Type below to see the last key pressed:

Last key: {{ lastKeyPressed }}

3. Key Modifiers (keyup.enter)

By appending .enter, Angular only triggers the method on the Enter key.

Type a message and press Enter to save:

Saved: {{ savedMessage }}

4. Mouse Events (mouseenter) / (mouseleave)

React to mouse hover states easily without CSS.

{{ hoverStatus }}

5. Form Events (focus) / (blur) / (input)

Track input changes and focus states as they happen.

Status: {{ focusStatus }}

Current Value: {{ inputValue }}

6. Global Events (window:resize)

You can bind to global targets like window or document directly from the template!

Resize your browser window to see this update:

Window width is: {{ windowWidth }}px

7. Custom Events output()

You can emit your own custom events for parent components to listen to.

(Check the console or bind to (myCustomEvent) in a parent component)
